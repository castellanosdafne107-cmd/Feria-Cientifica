# 🧬 Feria Científica 2026 - Ciencia Sin Límites

Sitio web oficial de la Feria Científica 2026 del Colegio Visión de Fe.

## 🎨 Paleta de Colores

| Color | Hex | Uso |
|-------|-----|-----|
| Azul Marino | `#0A2540` | Textos, fondos, seriedad |
| Menta | `#00D4AA` | Acentos, botones, brillo |
| Blanco | `#FFFFFF` | Fondos, limpieza |
| Gris Claro | `#F5F7FA` | Separación de secciones |

## 🔤 Tipografía

- **Títulos:** Plus Jakarta Sans (Bold)
- **Textos:** Inter (Regular)

## 🚀 Comandos

```bash
# Instalar dependencias
npm install

# Desarrollo
npm run dev

# Generar sitio estático
npm run generate

# Preview
npm run preview
```
